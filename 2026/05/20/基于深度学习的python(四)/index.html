张量

数组与张量

Pytorch将Numpy数组的语法金属吸收，作为自己处理张量的基本语法，且运算速度从使用CPU的数组进步到使用GPU的张量

Numpy和Pytorch的基础语法几乎一致，具体表现为

  • np对应 torch
  • 数组 array对应张量 tensor
  • Numpy的n维数组对应Pytorch的n阶张量，数组与张量之间可以互相转换
  • 数组 arr转为张量 ts；ts = torch.tensor(arr)
  • 张量 ts转为数组 arr；arr = np.array(ts)
